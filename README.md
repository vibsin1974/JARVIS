# JARVIS Interface

A JARVIS-inspired landing page with advanced animations and interactive elements.

## How to Run Locally

### Option 1: Using the provided script (Windows)
Double-click `run_local.bat` to start a local server and open the page.

### Option 2: Python
If you have Python installed, run:
```bash
python -m http.server
```
Then open `http://localhost:8000` in your browser.

### Option 3: Direct Open
You can simply open `index.html` in your web browser, though some features might be limited due to browser security restrictions on local files.
